Design Document Personal Project

During this semester I developed a portfolio to present my work. This portfolio has to be a website that I didn’t create with a tool (like Weebly, Google Sites or Wix), but that I developed using HTML and CSS. As this is a personal portfolio, I’d like the portfolio to show at least some design elements that have to do with me as a person.

Conclusion

I really wanted to put much more time into this project. The group project and the many deliverables that I had to take over from other people made this very hard. We were supposed to have personal project days on Friday, but all Fridays where spent on group project work. While I’m happy with what I’ve accomplished in such a short time, I’d have loved to work more on this project as I wanted to build a completely new website, especially after doing Open Learning for so many semesters. But I guess there’s nothing to do about that now.
